# Hackathon SMIT Campus Portal

A responsive Campus Portal web app for students to log in, report lost & found items, and manage campus resources.

---

## Features

- **User Login Page** with full-screen background image and modern card UI  
- **Lost & Found Reporting** to submit lost or found items  
- **Interactive Item Cards** with details, status badges, and action buttons  
- **Responsive Design** suitable for desktop and mobile devices  
- **Tailwind CSS** for quick and clean styling  
- **Customizable Assets** (logo, background image, item images)

---

## Tech Stack

- HTML5  
- CSS3 & Tailwind CSS  
- JavaScript (for form handling and modal pop-ups)  
- Git & GitHub (for version control)
